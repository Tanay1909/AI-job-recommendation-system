# AI Job Recommendation System

## Overview

A web-based application that recommends relevant job opportunities by analyzing user resumes using Machine Learning and Natural Language Processing.

---

## Features

* Resume parsing (PDF)
* Skill extraction using NLP
* Job recommendation with scoring
* Flask-based web interface

---

## Tech Stack

Python, Flask, scikit-learn, spaCy, Pandas

---

## Setup

```bash
git clone https://github.com/Tanay1909/AI-job-recommendation-system.git
cd AI-job-recommendation
pip install -r requirements.txt
python -m app.app
```

Access at: http://127.0.0.1:5000

---

## Deployment

Deployable on platforms like Render using `gunicorn app.app:app`.
